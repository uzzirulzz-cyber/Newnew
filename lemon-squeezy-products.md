## How to add each product in Lemon Squeezy

1. Go to https://app.lemonsqueezy.com/products → **New product**
2. Copy the **Name** and **Description** from below
3. Set **Price** = the PKR amount
4. Set **Pricing model** = Subscription (for monthly/yearly) or Single payment (for one-time)
5. Set **Tax category** = Software as a service (SaaS) for subscriptions, or Digital product for one-time
6. Set **Status** = Published
7. Enable **Display product on storefront** = Yes
8. Save — the product appears on your storefront within 60 seconds
